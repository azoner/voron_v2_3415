Recurring tasks to keep the printer happy.

## Cadence

| Task | Interval |
|---|---|
| Belt tension check | Every ~100 print hours |
| Linear rail lube | Every ~200 print hours, or after dust events |
| Z calibration / mesh refresh | Weekly, or when build plate / probe is disturbed |
| Nozzle cleaning | Every print start (PRINT_START handles it) |
| Filter change (CF / activated carbon) | Every ~500 print hours |
| Belt inspection (cracks, wear) | Every 6 months |

## Logs

Record incidents and resolutions in [Notes](../notes/) so the timeline is reviewable.
